Installation Instructions for DuelFieldStars
============================================

Requirements:
	* Python 2.7
	* Pygame

Steps:
	1) Download source and place in a directory (Extract if needed)
	2) Run export_svg.py from its directory (working directory is important)
	python2.7 export_svg.py
	3) Run duelfieldstars/main.py from its directory
	cd duelfieldstars; python2.7 main.py
	Report all bugs and crashes to github!
